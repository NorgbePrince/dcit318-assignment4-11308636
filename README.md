DCIT 318 – Assignment 4
MedicalDB SQL Database Project

Overview
This project is part of DCIT 318 Assignment 4.
It involves creating and managing a simple medical database using Microsoft SQL Server and SQL Server Management Studio (SSMS).

The database stores information about:

Patients (personal details)

Doctors (specialty and contact info)

Appointments (linking patients and doctors with appointment dates and reasons)

Features Implemented
Database Creation – Created MedicalDB.

Table Creation – Added Patients, Doctors, and Appointments tables with constraints.

Data Insertion – Inserted sample data into all tables.

Queries – Performed SELECT statements to retrieve data and show relationships between tables.

Update – Updated a patient’s phone number.

Delete – Deleted an appointment record.

How to Run
Open SQL Server Management Studio (SSMS).

Create a new query.

Paste the SQL script from the assignment steps.

Press F5 to run.

Tools Used
Microsoft SQL Server

SQL Server Management Studio (SSMS)
